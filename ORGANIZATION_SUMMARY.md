## 📝 Quick Reference Cards

### For explaining Backend
- Architecture: MVC pattern with 3-tier separation
- Authentication: BCrypt password hashing + JWT tokens
- Persistence: Spring Data JPA with H2/PostgreSQL
- Real-time: WebSocket with STOMP protocol

### For explaining Frontend
- Framework: React 18
- State: Component-level state management
- Styling: CSS modules per component
- API: Fetch for REST, STOMP for WebSocket

### For explaining Database
- Users table: Stores user accounts
- Messages table: Stores chat messages
- One-to-many: Users → Messages relationship
- Constraints: Unique username/email, NOT NULL fields
